# Churn-Prediction-using-ANN
## Final Year Project AI Data Science

### Project Objective 

Deep learning models like Artificial Neural Networks (ANNs) effectively predict e-commerce customer churn by identifying complex, nonlinear patterns in large datasets (e.g., usage behavior, transaction history).

# Key Steps in Building a Model
## Data Preprocessing: 
   - Cleaning data, handling missing values, and scaling features for neural networks.
## Training: 
   - Building an ANN to learn from historical customer data and minimizing loss over epochs.
## Evaluation: 
   - Testing accuracy on unseen data to ensure the model generalizes well.
